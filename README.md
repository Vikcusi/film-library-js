## Кинотека

Удобный каталог фильмов, в который можно добавлять карточки с названием, жанром и годом выпуска, отмечать просмотренные фильмы, а также редактировать и удалять записи. Поддерживается сортировка по названию, году выпуска и жанру.

[https://vikcusi.github.io/film-library-js/](https://vikcusi.github.io/film-library-js/)
